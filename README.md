# Photo App API

## Documentation

For API design, please vist [https://photos14.docs.apiary.io](https://photos14.docs.apiary.io)

## Prerequisites

- Python 3.7
- MySQL 5.7

## Quick start guide

1. Clone the repo

```
$ git clone git@github.com:duong-le/flask-photo-app.git
$ cd flask-photo-app
```

2. Start the server:

```
$ docker compose up
```

3. Navigate to [http://localhost:5000](http://localhost:5000)
